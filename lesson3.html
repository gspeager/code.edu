Lesson 3: Comparison Operators

Material

Comparison operators work much like the mathematical operators from lesson 1 except they return a result called a boolean. These comparison operator results can either be true or false, which allows a programmer to make decisions. There are many useful operators to know, and they are all used to determine the equality or difference between values.

  • Console Help
  • Next Lesson
Part 1: Equality

One of the most important comparison operators is to find out is something is equal to a specific value. The equality operator is == and will return true if both values are equal and false otherwise.

Action In the console compare two numbers with the equality operator.

Try entering something like:

3 == 3 

^\s*(?:-[0-9]+|[0-9]+)\s*==\s*(?:-[0-9]+|[0-9]+)
boolean
Part 2: Greater Than, Less Than

When using comparison operators, they compare from left to right. So just like in elementry school we can if a value is greater > or less < than another value.

Note these check that the values are exclusively larger than or smaller than the value being checked against.

Action In the console use a greater or less than operator to perform a comparison with two numbers.

Try entering something like:

2 < 4 

^\s*(?:-[0-9]+|[0-9]+)\s*[<>]\s*(?:-[0-9]+|[0-9]+)
boolean
Part 3: Or Equal To

Sometimes when checking values we want to ensure the value is equal or larger (or smaller) and not just strictly larger. This can be done by using the >= operator to check if it is greater than or equal to. The <= operator checks if the value is less than or equal to.

Action In the console use the less than or equal to or greater than or equal to operator with two numbers.

Try entering something like:

3 >= 2 

^\s*(?:-[0-9]+|[0-9]+)\s*(?:>=|<=)\s*(?:-[0-9]+|[0-9]+)
boolean
Part 4: Not Equal

When comparing values sometimes it is useful to simply check that a value is not equal to a specific value. We can do this with the not equals operator !=. If the two values are equal it will return false and true otherwise.

Action In the console use the not equal value with two numbers.

Try entering something like:

2 != 3 

^\s*(?:-[0-9]+|[0-9]+)\s*!=\s*(?:-[0-9]+|[0-9]+)
boolean
Part 5: Not Operator

We can also make a comparison return the opposite, true if it was false false if it was true. This can be done like !(comparison).

Action In the console use the not operator with a greater than comparison.

Try entering something like:

!(3 > 2) 

^\s*!\s*\(\s*(?:-[0-9]+|[0-9]+)\s*>\s*(?:-[0-9]+|[0-9]+)\s*\)
boolean
Part 6: AND Operator

The AND operator allows a programmer to combine two comparisons and return a single result when they both return true. Using the && operator between two comparisons will return the combined result.

Note AND returns true when both comparisons are true and false otherwise.

Action In the console use AND with two comparisons.

Try entering something like:

2 > 3 && 3 < 4 

^\s*(?:-[0-9]+|[0-9]+)\s*(?:>=|<=|==|!=|<|>)\s*(?:-[0-9]+|[0-9]+)\s*&&\s*(?:-[0-9]+|[0-9]+)\s*(?:>=|<=|==|!=|<|>)\s*(?:-[0-9]+|[0-9]+)
boolean
Part 7: OR operator

The OR operator works slightly different the AND operator. By using the || operator we can check if either of the comparisons are true. So it will return false only wen both comparisons are false and true otherwise.

Action In the console use the OR operator with two comparisons..

Try entering something like:

2 > 3 || 3 < 4 

^\s*(?:-[0-9]+|[0-9]+)\s*(?:>=|<=|==|!=|<|>)\s*(?:-[0-9]+|[0-9]+)\s*\|\|\s*(?:-[0-9]+|[0-9]+)\s*(?:>=|<=|==|!=|<|>)\s*(?:-[0-9]+|[0-9]+)
boolean

How to Use the Console

Enable the Console

To enable the console you must first click on the grey console area. You can tell that the console is enabled when the border around it gets darker and thicker. The cursor and line input marker will also get darker to let you know you can type.

Note clicking outside the console will disable it.

Using the console

Once the console is enabled you will be able to type into the input line. This allows you to enter commands and have them processed. If your command is executed successfully the result will:

>> look like this
If your input results in an error it will:
look like this

Special Keys

While the console will register most character keys there are a few keys that perform special actions.

  • Enter submits the current input line
  • Backspace deletes the character to the left of the cursor
  • Delete deletes the character in the cursor
  • Left Arrow moves the cursor one character to the left
  • Right Arrow moves the cursor one character to the right
  • Up Arrow cycles backwards through previously used commands
  • Down Arrow cycles forward through previously used commands
