Lesson 2: Strings

Material

Strings are made up of characters and are usually used to display text. An example of a string is 'Hello World'. Strings are also extremely important to understand because they represent text and being able to manipulate text can be invaluable. There are many different ways to get information from strings and modify them to suit your uses.

Part 1: Characters

A string is made up of characters. Characters can be simple letters from the alphabet such as c or can be specialized characters like #.

Action In the console type some characters.

Try entering something like:

Hello World 

.+

Part 2: String Format

You'll notice that the console didn't like characters by themselves. For a string to be recognized, you must surround your characters with either " or '.

You can not mix and match the quotation marks so if you start your string with a " make sure you end it, with the same one.

Action In the console enter a string.

Try entering something like:

"Hello World" 

["'].+["']
string
Part 3: String Concatenation

Now that you understand what a string is, there are many things you can do with it. Concatenation is one of the most common operations performed with strings. Concatenation combines two strings together with the + operator.

Action In the console try and concatenate two strings together.

Try entering something like:

"Hello" + " World" 

\+
string
Part 4: Length

Sometimes it is extremely important to know how long a string is. It can allow us to perform operations using each character if we need to. Using .length immediately after the string we can easily figure out how long a string is.

Action In the console get the length of your name.

Try entering something like:

"Hello World".length 

["'].+["'].length
number
Part 5: Uppercase and Lowercase

Now we can get into string functions. There are many useful string functions to manipulate strings in different ways. Two in particular are .toUpperCase() and .toLowerCase(). As you can probably guess, these will convert all the characters in a string to uppercase or lowercase depending on which you choose. You also notice that unlike .length these functions end with brackets, you must include those for it to work.

Action In the console try using one of the case functions with a string.

Try entering something like:

"Hello World".toUpperCase() 

["'].+["'].toUpperCase\(\)|["'].+["'].toLowerCase\(\)
string
Part 6: Replace

Another useful function is .replace(). It takes two parameters. In this case you can use it as .replace("string to replace", "string to replace it with").

Action In the console use replace to modify a string.

Try entering something like:

"Hello World".replace(" ", "_") 

["'].+["'].replace\(.+\)
string
Part 7: IndexOf

A simpler string function is .indexOf(). This function takes a single string parameter and returns the position of the first found occurrence of the parameter in the string. Alternatively you can also use .lastIndexOf() to get the last occurrence of a string.

Note In programming, most structures start at position 0, so if you used .indexOf() with the first character in the string, it would return location 0 and the second character would return 1.

Action In the console use either of these functions to get the position of a character.

Try entering something like:

"Hello World".indexOf("o") 

["'].+["'].indexOf\(.+\)|["'].+["'].lastIndexOf\(.+\)
number
Part 8: Substrings

Finally, one of the most useful functions to use with strings is .substr(). It takes two numbers as parameters. The first is the start postion and the second is how many characters to extract.

Action In the console extract a substring from a string.

Try entering something like:

"Hello World".substr(1,4) 

["'].+["'].substr\(.+\)

How to Use the Console

Enable the Console

To enable the console you must first click on the grey console area. You can tell that the console is enabled when the border around it gets darker and thicker. The cursor and line input marker will also get darker to let you know you can type.

Note clicking outside the console will disable it.

Using the console

Once the console is enabled you will be able to type into the input line. This allows you to enter commands and have them processed. If your command is executed successfully the result will:

>> look like this

Special Keys

While the console will register most character keys there are a few keys that perform special actions.

  • Enter submits the current input line
  • Backspace deletes the character to the left of the cursor
  • Delete deletes the character in the cursor
  • Left Arrow moves the cursor one character to the left
  • Right Arrow moves the cursor one character to the right
  • Up Arrow cycles backwards through previously used commands
  • Down Arrow cycles forward through previously used commands
