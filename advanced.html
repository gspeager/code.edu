Conditionals

Often when writing code you will want to change the behaviour of the program based on values from inputs and calculations. This can be accomplished by using comparison operators with conditional statements. We the different types of conditional statements are:

  • if statement - use this statement to execute some code only if a specified condition is true
  • if...else statement - use this statement to execute some code if the condition is true and another code if the condition is false
  • if...else if....else statement - use this statement to select one of many blocks of code to be executed
  • switch statement - use this statement to select one of many blocks of code to be executed

If Statements

If statments are used to check a comparison and then perform a section of code if the comparison evaulates to true. The syntax for an if statement is: 

if(comparison statement) {
	//Perform code
}

If Else Statements

If Else statements are an extension of the if statement where there is a section of code to perform if the comparison evaluates to true. Although, if the comparison evauluates to false there is a second section of code to be executed. 

if(comparison statement) {
	//Perform code if true
}
else {
	//Perform code if false
}

Else If Statements

Else if statements work with the If and Else statements but allow for a multiple comparison checks before settling in the Else block. Else if can be used once or more with a comparison block. 

if(comparison statement) {
	//Perform code if true
}
else if (comparison statement two) {
	//Perform code if true
}
else {
	//Perform code if false
}

Switch Statments

Swtich statements are slightly different then if statements. Switch statements do not rely on a comparsion operation, but accept a value and then run through a list of potential values to perform actions for, and then if there is no match a default section of code can be run. 

switch(value) {
	case value1:
  		//Perform code if equal
  		break;
 	case value2:
  		//Perform code if equal
  		break;
  	case value3:
  		//Perform code if equal
  		break;
  	default:
  		//Perform code if no matches
}

Loops

Loops allow a programmer to repeatedly perform a section of code until a specific condition is met to break the cycle. There are three basic types of loops

  • While loop
  • Do while loop
  • For loop

While Loop

A while loop is the simplest form of a loop in programming. It has a single condition, and until the condition is no longer true it will run through the code inside the loop. 

while(comparison statement is true) {
	//Perform code
}

Do While Loop

The do while loop is similar to the while loop except it performs the loop code before performing the check. It is essentially the same as if you flipped the while loop. 

do{
	//Perform code
}while(comparison statement is true)

For Loop

For loops are similar to while loops, but attempt to consolidate the initialization of a counter variable for how many times the loops should be run.

A for loop requires three statements to set it up. The counter initialize statement, the conditional to stop the loop, and the increment for counter after each time the loop is ran through.

Note The conditional and increment can be set to to any valid operations or statements. 

for(counter=val;counter<limit;counter++) {
	//Perform code
}

Loops and Arrays

Loops are great for evaluating and using arrays. Using a for loop you can set it up to evaluate an array very easily. 

for(index=0;index<myArray.length;index++) {
	sum += myArray[index];
}

Associative Arrays

Associative arrays are very similar to the standard arrays covered in lesson 5, but with one difference. They use strings as the index instead of numbers, that way values in the array can be associated with a descriptive string instead.

Associative arrays are set up slightly different: 

var daysInMonth = new Array();
daysInMonth["January"] = 31;
daysInMonth["February"] = 28;
daysInMonth["March"] = 31;
...

Then when we want to access the days for a sepecific month we just need to enter the month as a string into as the index. If we enter something like daysInMonth["January"] we will get 31 back. Beyond that the array can be manipulated in a similar fashion to the standard arrays.

Functions

Functions are a small or large piece of executable code which can be ran by calling the shorthand name for the function. Functions can take parameters to modify the processing inside the code block, and also can have a return value which comes out at the end of the function. An outline of a function can look like: 

function functionname(var1,var2,...,varX) {
	//Code to execute
	(optional) return val;
}

Using that outline we can create our own simple function: 

function hello(name) {
	return "Hello " + name;
}
Anytime we would need a string to say hello we could use our function like result = hello("Mike"). Then when we use the variable result it will contain "Hello Mike".

Note Functions should only accomplish a single task and can be as complicated or simple as is necessary to complete it.

Classes

Classes are a structure which contain variables and functions which are related in some way. An example of a class is: 

function Person(firstName,lastName,age) {
	this.firstName = firstName;
	this.lastName = lastName;
	this.age = age;

	this.firstAndLastName = function() {
        return this.firstName + " " + this.lastName;
    }
}
This can look similar to a function, but by having having the variables defined with this. it will allow us to access the variables. When using a class we have to apply it to a variable with the new keyword. If we want to make a new person it will look like:

myVariable = new Person("John", "Smith", 34)

Once that is defined we can access the variables and functions of the class extremely easily: 

myVariable.firstName 		//returns "John"
myVariable.firstName 		//returns "Smith"
myVariable.age 			//returns 34
myVariable.firstAndLastName()	//returns "John Smith"

When defining variables inside a class, instead of this. we can just use var . That means the variable will be called private and only accessible to functions inside of the class.

Other Languages

There are many, many other lanuages to learn which can suite different purposes. Some of the more prominent languages used for development today are:

Java

Java is probably the most popular language in the world right now and is especially used in academic and research circles. It requires code to be compiled and then executed. Java requires the Java Runtime Environment to be installed on computers which need to run java programs. Since Java uses its own runtime environment, it allows the code to run the same on any operating system. Unfortunately, executing the program through the Java Runtime environment can make the program run pretty slow depending on what operations it is performing.

C/C++

These are the fastest, lightest modern compiled languages available right now. It executes incredibly fast, and programs can be written with minimal extra content included. This makes it an ideal choice for computers which have very restricted hardware. The only downside of using these languages is that because they are so light weight, lots of functions are not particularly programmer friendly, and require a lot of extra work.

C#

Developed by Microsoft, C# is supposed to be the next evolution of C++. It does a lot more work for the programmer, making things easier and quicker to develop. These bonuses come with a cost, it requires the .NET framework to execute any applications. The .NET framework is installed on any Windows computer, but may require an update depending on the program. Even though it is much easier to program with, the .NET framework makes applications run slightly slower when compared to C++ applications, and restricts the program's platforms to Windows PCs.

Python

Python is a scripting langauge based on C++. This means that it does not require to be compiled. A python interpreter must be installed on a PC needing to run phython scripts, but then the script can be immediately executed. Python also has really simple structures and uses, so that writing programs can be extremely quick. It is a nice option for small programs because it executes relatively fast for non-compiled applications and is very quick and easy to write.

Ruby

Ruby is very similar to Python but slightly slower. It uses slightly different programming syntax, but does not require compilation either. Ruby is a relatively new language, but has a huge development community. This means that Ruby has lots of documentation available to learn and resolve problems, as well as lots of community developed, open source, plugins to use which makes developing applications extremely fast.
