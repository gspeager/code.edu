Lesson 5: Arrays

Material

When programming arrays are basically variables which can hold multiple values. An array can be used to store multiple numbers or strings which have some kind of relationship. The different elements inside an array can be accessed by using an index, which represents the value's position inside the array. This way, a collection of values can be stored and manipulated similarly to variables, while maintaining the structure.

Part 1: Defining Arrays

Arrays have the same require the same naming conventions as variables. But how do we define multiple values in the same variable? Well we use square-brackets and and separate the values by commas.

Arrays can be defined as:
myArray = [val1,val2,val3,...]

Action In the console create an array with four numbers.

Try entering something like:

myArray = [2,4,6,8] 

^\s*[a-zA-Z_][a-zA-Z_0-9]*\s*=\s*\[[0-9]+,\s*[0-9]+,\s*[0-9]+,\s*[0-9]+\]
object
Part 2: Indexes

To access the data stored in arrays you need to use an index with your array. Array indexes start at 0 and go up to one less then the total number of values in the array. Since we created an array with four numbers, our indexes would be 0-3.

Indexes can be used like myArray[index]. We use the [] to let the variable know that we want to use the value at the position defined inside the brackets.

Action In the console access one of the values from your array.

Try entering something like:

myArray[2] 

^\s*[a-zA-Z_][a-zA-Z_0-9]*\[[0-9]+\]
number
Part 3: Using Array Values

Once we can access values in arrays they can be used just like any other variable. Since our array contains numbers we can use any number operations on the elements of the array. If the array contained strings we could use the string functions.

Action In the console add two array values together.

Try entering something like:

myArray[2] + myArray[1] 

^\s*[a-zA-Z_][a-zA-Z_0-9]*\[[0-9]+\]\s*\+\s*[a-zA-Z_][a-zA-Z_0-9]*\[[0-9]+\]
number
Part 4: Array Assignment

When changing the values inside an array you have to ensure that you use an index to change a specific value in the array. This is done as myArray[index] = newVal.

If an index isn't used like myArray = value you will lose the entire array, and will only be able to reference the value like a normal variable. This can be used to overwrite an array with new values if you use the array definition from part 1.

Action In the console overwrite one of the values in your array.

Try entering something like:

myArray[2] = 3 

^\s*[a-zA-Z_][a-zA-Z_0-9]*\[[0-9]+\]\s*=\s*(?:-[0-9]+|[0-9]+)
number
Part 5: Array Length

Like with strings arrays also have a .length property, which returns how many values are contained within the array.

Note this is extremely useful to get the last value of the array. You can easily access it with myArray[myArray.length - 1].

Action In the console get the length of your array.

Try entering something like:

myArray.length 

^\s*[a-zA-Z_][a-zA-Z_0-9]*\.length
number
Part 6: Adding More Values

It is easy to add values to your array and make it longer. All you need to do is select an index larger then your current array and assign a value to it. The easiset way to add on to the end of your array is to use .length as the index, since it will return a value one larger then your largest index.

Note If you have an array with 50 values, you will have to use indexes 0-49 to access those values. Then if you decide to add a new value at index 100, you will have indexes 50-99 with no values. That could make keeping track of your data extremely difficult.

Action In the console add a value to the end of your array using .length.

Try entering something like:

myArray[myArray.length] = 10 

^\s*[a-zA-Z_][a-zA-Z_0-9]*\[\s*[a-zA-Z_][a-zA-Z_0-9]*\.length\]\s*=\s*(?:-[0-9]+|[0-9]+)
number
Part 7: Multidimensional Arrays

Multidimensional arrays can be a confusing concept, but we mostly know how to work with them already! If we create a new array with arrays as the values, then we can have what is called a two dimensional array. The syntax for this is myArray = [[val1, val2, val3,...],[val1, val2, val3,...],...]

Action In the console create an array which contains two arrays of four numbers.

Try entering something like:

my2dArray = [[2,4,6,8],[9,8,7,6]] 

^\s*[a-zA-Z_][a-zA-Z_0-9]*\s*=\s*\[\s*\[(?:-[0-9]+|[0-9]+),\s*(?:-[0-9]+|[0-9]+),\s*(?:-[0-9]+|[0-9]+),\s*(?:-[0-9]+|[0-9]+)\s*]\s*,\s*\[(?:-[0-9]+|[0-9]+),\s*(?:-[0-9]+|[0-9]+),\s*(?:-[0-9]+|[0-9]+),\s*(?:-[0-9]+|[0-9]+)\s*]\]
object
Part 8: Accessing Values in Multidimensional Arrays

Before we used indexes to access values in an array, well that is still the case. The only trick now is that an index will retrieve our array, then we need a second index to retrive the values from that array. This is easily done by myArray[index1][index2]

Note we can use this same method to create arrays with many more dimensions then two, and they all can be accessed by adding more indexes.

Action In the console retrieve a value from the two dimensional array.

Try entering something like:

my2dArray[0][2] 

^\s*[a-zA-Z_][a-zA-Z_0-9]*\[[01]+\]\s*\[[0-3]+\]
number

How to Use the Console

Enable the Console

To enable the console you must first click on the grey console area. You can tell that the console is enabled when the border around it gets darker and thicker. The cursor and line input marker will also get darker to let you know you can type.

Note clicking outside the console will disable it.

Using the console

Once the console is enabled you will be able to type into the input line. This allows you to enter commands and have them processed. If your command is executed successfully the result will:

>> look like this
If your input results in an error it will:
look like this

Special Keys

While the console will register most character keys there are a few keys that perform special actions.

  • Enter submits the current input line
  • Backspace deletes the character to the left of the cursor
  • Delete deletes the character in the cursor
  • Left Arrow moves the cursor one character to the left
  • Right Arrow moves the cursor one character to the right
  • Up Arrow cycles backwards through previously used commands
  • Down Arrow cycles forward through previously used commands
