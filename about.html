About Code.Edu

Information

This was created as a reusable learning object by Garrett Speager to teach the basics of programming using javascript. The source will be hosted on github so that the code will be available to everyone to modify and learn from. The site will also be hosted using github pages to make Code.Edu available on the web.

Tools

Developed with HTML and Javascript using the jQuery library.

Used Sublime Text 2 for text editing, Firefox with Firebug and Chrome to test, and TortoiseGit for Github.
