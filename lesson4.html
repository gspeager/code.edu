Lesson 4: Variables

Material

Junior high mathematics taught us that variables can be used in place of unknown value in an equation. They can also be used in equations for values that can change, therefore giving different results. Variables are eseential to understand in programming because you will use them for everything when programming. In a computer program variables are a set of characters which represent a value and can be set, modified, and changed.

Part 1: Declaring Variables

In many programming lanuages variables can be declared with a data type so that the program knows how much memory is required to store the information that variable. In this language things are much simpler, you only need to define a name for your variable and assign a value to it such as myVariable = 2. When you give a variable a value it is initialized.

Variable names are case sensitive so x and X would be two separate variables. Names must also start with a character or '_'.

Action In the console initialize a variable with a number.

Try entering something like:

myVariable = 2

Part 2: Using Variables

Once you have initialized a variable you can use any of the operations or functions available with that type. So if your variable is a number, all the mathematical operations can be used. If your variable is a string then you can use the functions you learned in the previous lesson directly on the variable.

Action In the console add a number with your new variable.

Try entering something like:

myVariable + 3

Don't forget that myVariable must be initialized with a value first, which you did in part 1.

Part 3: Assignment Operator

Variables use one unique operator which numbers on their own cannot. With variables you can use the = operator, like when initializing a variable, to change the value of a variable at any time.

Note Assignment operators always work from right to left. The value on the right always goes into the variable on the left. Assignments can be chained like x=y=2, which makes both x and y contain the value 2.

Action In the console assign a new number to your variable.

Try entering something like:

myVariable = 3+2 

^\s*[a-zA-Z_][a-zA-Z_0-9]*\s*=\s*.+
number
Part 4: Number Increment

Often when using variables with numbers you will want to just increase the value of a variable by one. Shorthand operators have been created to easily increment variables. Using ++ with a variable will increment it by one, and there are two ways to use it:

  • Before the variable: such as ++myVariable - if the variable is being used for anything in the line where it is incremented, it will increment the variable FIRST then perform any calculations with the variable.
  • After the variable: such as myVariable++ - if the variable is being used for anything in the line where it is incremented, it will perform any calculations with the variable then increment the value AFTER the line is complete.

Note you'll notice if you use the ++ after the variable the the output from the command wont look like it was incremented, this is because the increment happens after calculations are complete. If you type in your variable name a second time you will see that the value was indeed updated.

Action In the console increment your variable.

Try entering something like:

myVariable++ 

\+\+[a-zA-Z_][a-zA-Z_0-9]*|^\s*[a-zA-Z_][a-zA-Z_0-9]*\+\+
number
Part 5: Number Decrement

Number variables can also be decremented exactly like the increment, the only difference is that you use -- instead of ++.

Action In the console try decrementing your variable.

Try entering something like:

myVariable-- 

--[a-zA-Z_][a-zA-Z_0-9]*|^\s*[a-zA-Z_][a-zA-Z_0-9]*--
number
Part 6: Special Assignment Operators

Another common operation with number variables is to make the variable equal to some calaculation based on its current value. There have been shorthand assignment operators to accompilsh this very thing. Instead of typing out something like myVariable = myVariable + 3 the shorthand version can be used with less typing myVariable += 3. This is available for all mathematical operators:

  • Addition: +=
  • Subtraction: -=
  • Multiplication: *=
  • Division: /=
  • Modulo: %=

Action In the console use multiplication assignment operator with your variable.

Try entering something like:

myVariable *= 3 

^\s*[a-zA-Z_][a-zA-Z_0-9]*\s*\*=\s*(?:-[0-9]+|[0-9]+)
number
Part 7: Other Variables

Variables are not resticted to being used with just numbers in programming. You can create a variable with any object or type. When you create a variable with a type all the functions and operators will become available to the variable. This means we can also create variables to represent strings.

Action In the console create a variable with a string.

Try entering something like:

myVariable2 = "Hello" 

^\s*[a-zA-Z_][a-zA-Z_0-9]*\s*=\s*["'].+["']
string
Part 8: Length Revisited

Just like with numbers any functions and operators we used with strings are available to a variable containing a string.

Action In the console get the length of the string contained in your variable.

Try entering something like:

myVariable2.length 

^\s*[a-zA-Z_][a-zA-Z_0-9]*\.length

How to Use the Console

Enable the Console

To enable the console you must first click on the grey console area. You can tell that the console is enabled when the border around it gets darker and thicker. The cursor and line input marker will also get darker to let you know you can type.

Note clicking outside the console will disable it.

Using the console

Once the console is enabled you will be able to type into the input line. This allows you to enter commands and have them processed. If your command is executed successfully the result will:

>> look like this
If your input results in an error it will:

Special Keys

While the console will register most character keys there are a few keys that perform special actions.

  • Enter submits the current input line
  • Backspace deletes the character to the left of the cursor
  • Delete deletes the character in the cursor
  • Left Arrow moves the cursor one character to the left
  • Right Arrow moves the cursor one character to the right
  • Up Arrow cycles backwards through previously used commands
  • Down Arrow cycles forward through previously used commands
